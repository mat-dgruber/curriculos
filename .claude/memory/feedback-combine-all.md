name: Combine All Approaches
description: When given multiple approaches for a feature, user prefers combining all rather than choosing one
type: feedback
---

When presented with multiple feature approaches, user prefers combining all options rather than picking one.

**Why:** User sees value in every approach and doesn't want to lose functionality. "Vamos unir todas" was said about job exclusion methods, and "os dois juntos" about matcher improvement (refine scrapers AND matcher).

**How to apply:** When brainstorming features with discrete options (A, B, C), default to proposing A+B+C combined rather than asking the user to pick one. Only force a choice when options are genuinely mutually exclusive.
